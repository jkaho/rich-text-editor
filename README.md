# Rich Text Editor 

## Description 

This is a basic rich text editor that allows users to apply formatting to text. The formatting features that are currently supported are: 
- Font
- Font size 
- Bold 
- Italic
- Underline 
- Unordered lists
- Ordered lists

## Technologies Used

- JavaScript
- HTML
- CSS
- Font Awesome
